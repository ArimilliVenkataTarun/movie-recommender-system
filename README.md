# 🎬 Movie Recommendation System

A **content-based movie recommendation system** built using Python and Machine Learning.
This project recommends movies similar to a selected movie based on genre similarity.

---

## 🚀 Features

* 🎯 Movie-to-movie recommendations
* 🧠 Content-based filtering
* ⚡ Fast similarity computation
* 🖥️ Simple and clean Python implementation
* 📊 Uses cosine similarity for accuracy

---

## 🧠 How It Works

1. Movie genres are converted into numerical vectors using **CountVectorizer**
2. **Cosine similarity** measures similarity between movies
3. The system recommends the most similar movies to the selected input

---

## 🛠 Tech Stack

* **Python**
* **Pandas**
